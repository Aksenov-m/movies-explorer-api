## movies-explorer-api
Репозиторий для приложения бэкенд части проекта `movies-explorer-api` дипломного проекта яндекс практикум. 

## Запуск проекта

`npm run start` — запускает сервер  
`npm run dev` — запускает сервер с hot-reload
